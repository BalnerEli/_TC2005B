¿Qué diferencias y semejanzas hay entre Java y JavaScript?
Diferencias
1. Lenguaje:Java es un lenguaje de programación orientado a objetos y compilado, mientras que JavaScript es un lenguaje de scripting interpretado , que se usa para navegadores web
2. Aplicación: Java se utiliza para desarrollo de aplicaciones móviles, aplicaciones de escritorio, sistemas embebidos, etc. JavaScript se usa para el desarrollo web y en entornos de servidor como Node
3. Compilación: Java requiere una etapa de compilación en la que el código fuente se compila a bytecode y luego esta se ejecuta en la máquina virtual Java. JavaScript, en cambio, es interpretado directamente por el navegador o el entorno de ejecución no necesita de una compilación previa.
Semejanzas
1. Ambos utilizan funciones para encapsular bloques de código reutilizabl es.
2. En ambos, existen librerías y frameworks que facilitan el desarrollo

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
Pueden introducir errores y dificulta la legibilidad del código. Al poder modificar las variables en cualquier parte del código dificulta tener un control de modificaciones y por ende, dificulta la corrección de errores.
